# evaluation_of_job_housing_balance_for_key_workers
 ✨ Thesis

- About:
  в диссертации я изучаю показатель несоответствия работы и места проживания
  показатель рассчитывается достаточно просто: практически отношение аренды к зп умноженное (с коэффициентом) на удаленность работы.
  Показатель отражает комфортность проживания различных специалистов в разных районах города (и городе в целом).

  key workers -- достаточно широкое определение. В основном это наиболее необходимые городу работники, без которых город не может существовать (а.к.а бюджетники)
  В моем случае это скореее primary workers -- работники, которые нужны городу для развития.

- Цель работы:
  показать, что нужно развивать не только центр города и строить доступное жилье.

- Данные: 
  о домах (*росеестр*)
  о вакансиях (хх)
  о бизнесах/работодателях (хх + фнс)
  о городе (осм)
